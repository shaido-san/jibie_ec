                   +-----------------------+
                   |                       |
                   |      Client (Browser)  |
                   |                       |
                   +-----------+-----------+
                               |
                               |  クライアント（ブラウザ）からリクエストが送信される
                               |
                +--------------v--------------+
                |                             |
                |          Nginx Server       |
                |                             |
                +--------------+--------------+
                               |
                               |  Nginx がリバースプロキシとしてリクエストを受け取り、

                               |
                  +------------v------------+
                  |                         |
                  |   Gunicorn (WSGI Server)|
                  |                         |
                  +------------+------------+
                               |
                               |  Gunicorn が WSGI アプリケーション（Django）を処理する
                               |
                  +------------v------------+
                  |                         |
                  |    Django Application   |
                  |                         |
                  +------------+------------+
                               |
                               |  Django アプリケーションがビジネスロジックを実行し、
                               |  必要なデータをデータベースから取得・保存する
                               |
               +---------------v---------------+
               |                               |
               |    PostgreSQL Database (DB)   |
               |                               |
               +-------------------------------+
                               |
                               |  データベースがデータの読み書きを担当
